# Schilling's Content Agents
Multi-agent content automation system for Schilling's English Course.
Built with Node.js, Claude API, Express, Puppeteer and Evolution API.

## Agents
Copy · Posts · SEO · LinkedIn · Email · Funil · Ads · Design

## Bot
WhatsApp bot (Shelly) powered by Evolution API + Airtable

## Stack
Node.js · Claude Sonnet API · Express · Puppeteer · Airtable · Evolution API

## How to run
1. cp .env.example .env and fill in your keys
2. npm install
3. node main.js "your topic"
4. node ui/server.js → localhost:3000
5. node bot/server.js → localhost:3002
